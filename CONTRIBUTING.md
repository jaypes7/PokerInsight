# Contributing

This is a private project. Contributors must follow `AGENTS.md`, use task IDs
from `TODO.md`, and open pull requests with the repository template.

## Local Setup

1. Install Docker, Node.js 20, pnpm 9, Python 3.12, and uv.
2. Copy `.env.example` to `.env.local`.
3. Run `make install`.
4. Run `make dev`.

## Commits

Use Conventional Commits and include `Refs: <TASK-ID>` in the footer.
